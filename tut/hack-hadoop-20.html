Hack Hadoop-20

About

This is a tutorial series conducted by Runhui Li. After the tutorial you should understand:
  1. how Hadoop-20 use erasure coding to perform data recovery, degraded read, and
  2. how we can hack the erasure coding, block placement and MapReduce modules.
I will dive into the source codes and demonstrate how to hack the source codes to integrate our own system.

Materials

Contents

  1. Introduction and erasure coding routine [PPTX]
  2. Add your own coding scheme: toy example of RDP code + JNI integration
  3. Hacking MapReduce scheduler
  4. Hacking the placement scheme
  5. Benchmark and coding style
